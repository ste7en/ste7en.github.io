In this section are presented the software projects done during my Bachelor's Degree in Engineering of Computing Systems, with their grade and links to GitHub.

Adrenaline by Cranio Creations has its own page on this portfolio and is part of the Software Engineering 1 course, graded 30/30 cum Laude. You can find its page here.

SimpleFS is a simple UNIX-like file system written in C as project of the Algorithms and Data Structures course, graded 30/30.
You can see it on GitHub.

The Manhattan Distance Evaluator is a VHDL implementation of a digital circuit that evaluates which points (centroids) are closer to a given point. It is part of the course Reti Logiche (aka Digital Systems Design) and has been graded 30/30 cum Laude.
You can see it on GitHub.
